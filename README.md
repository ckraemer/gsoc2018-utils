# GSoC 2018 Utilities
This repository contains tools for the GSoC project [Implementation of a user space interface for GPIO interrupts](https://wiki.freebsd.org/SummerOfCode2018Projects/UserSpaceGPIOinterrupts).
The actual development of the driver takes place in [my fork of the FreeBSD src tree](https://github.com/ckraemer/freebsd/tree/gsoc2018).
